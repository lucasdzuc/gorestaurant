Para inicializar todos os testes já prepardos, utilize <code> yarn test </code>.

# Como rodar? 🤔
Siga os passos:

```
    yarn
    npm i -g json-server
    yarn server
    yarn start

```

# Como Contribuir?

```bash
# Clone o seu fork
